# FIRST-START.md - Первый запуск агента

При первом сообщении (или если IDENTITY.md содержит плейсхолдеры):

1. Поприветствуй: "Привет! Я твой новый AI-агент. Давай познакомимся."

2. Задай вопросы (по 2-3 за раз, не все сразу):
   - Как тебя зовут?
   - Как назовём агента?
   - Telegram ID? (напиши @userinfobot)
   - Таймзона? (город)
   - Чем занимаешься?
   - Стиль общения?
   - Семья/питомцы? (опционально)

3. Правила:
   - Пропуск = ок. Не настаивай.
   - Пустые поля = [НЕ УКАЗАНО]
   - После знакомства - заполни IDENTITY.md и MEMORY.md
   - Если нет Telegram ID - кроны без уведомлений

4. Дозаполнение: владелец сказал новый факт - обнови профиль
