# 📊 Стратегическая сессия - скилл для OpenClaw

AI-фасилитатор стратегических сессий на базе мировых практик: EOS, OKR, 4DX, Scaling Up.

## Что это

Скилл превращает вашего AI-агента в бизнес-коуча. Он проводит полноценную стратегическую сессию - с анализом метрик, разбором провалов (5 Whys), pre-mortem рисков, постановкой целей и фиксацией результатов.

Работает для соло-предпринимателей, фрилансеров, контент-криейторов и команд до 50 человек.

## Что внутри

- **SKILL.md** - основной файл скилла (положить в папку skills)

## Установка

### Вариант 1: Ручная установка

1. Создайте папку для скилла:
```bash
mkdir -p <ваш-workspace>/skills/strat-session
```

2. Скопируйте `SKILL.md` в эту папку:
```bash
cp SKILL.md <ваш-workspace>/skills/strat-session/SKILL.md
```

3. Перезапустите gateway:
```bash
openclaw gateway restart
```

4. Проверьте что скилл подхватился - напишите агенту "стратсессия" или "стратегическая сессия".

### Вариант 2: Через clawhub (когда будет доступно)

```bash
openclaw skill install strat-session
```

## Как работает

1. **Первый запуск** - агент проводит онбординг: узнаёт о вашем бизнесе, целях, метриках. Создаёт профиль.
2. **Каждая сессия** - собирает данные, показывает Scorecard, разбирает что сработало/не сработало, ставит цели на следующий период.
3. **Результат** - структурированный документ с Rocks (приоритетами), WIG (главной целью), lead measures и планом действий.

## Методологии

- **EOS** (Entrepreneurial Operating System) - Rocks, Scorecard, People Check
- **OKR** (Objectives & Key Results) - целеполагание с измеримыми результатами
- **4DX** (4 Disciplines of Execution) - WIG, Lead Measures, Scoreboard
- **Scaling Up** (Verne Harnish) - OPSP, SWT Analysis

## Опциональные интеграции

Скилл работает полностью автономно. Но если настроены - может использовать:
- `analytics` - автосбор метрик YouTube/Telegram
- `business-architect` - анализ unit economics
- `brainstorming` - генерация идей
- `gog` - сохранение в Google Docs/Sheets
